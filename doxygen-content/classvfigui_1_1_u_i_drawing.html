draw image data to screen

Parameters
[in]typeimage type
[in]dataimage data (format depends on type)
[in]sizenumber of bytes in data
[in]xx-coordinate
[in]yy-coordinate
[in]wwidth of the image, only used for IMG_RAW
[in]hheight of the image, only used for IMG_RAW
Note
In case of IMG_RAW the number of bytes per pixel is determined from size and w*h. Depending on the number of bytes per pixel the image format is considered to be as follows:
  • 1: Grayscale image
  • 2: Grayscale image with alpha
  • 3: RGB image
  • 4: RGB image with alpha
