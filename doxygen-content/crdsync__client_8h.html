◆ crdSync_ReadSecurityMem()

CRDSYNC_RET crdSync_ReadSecurityMem ( unsigned char  server_idx,
unsigned char  RxBuff[],
unsigned char *  BufLen 
)

Read Security Memory from synchronous card
.

Without a preceding successful crdSync_PresentSecretCode() only the first byte is read.
This represents the Error Counter (EC). Each set bit shows a remaining PSC entry try.
Max. PSC entry tries: 8 for SLE4428, 3 for SLE 4442.
With a preceding successful crdSync_PresentSecretCode() additionally the PSC is read.

Parameters
[in]server_idx- index of server in server configuration (see crdSync_ConfigureServer()).
[out]RxBuff- Pointer to buffer where security memory has to be stored
[in,out]BufLen- security memory length returned
Returns
CRDSYNC_Success - Operation successful
CRDSYNC_Failure - Error - driver not opened, card not inserted or powered up
CRDSYNC_Error_BadParam - Invalid parameter, TxLen is zero, or TxBuf is NULL
CRDSYNC_Error_Protocol - Error communicating with the card

SLE4428: Read data with protection bit
.

Read RxLen data bytes + 1 byte protection bit per data byte

Parameters

[in]MemoryAddress- memory address where to start reading from if -1, start at current location
[out]RxBuf- Pointer to buffer where the read data (incl. protection bits) has to be stored Must be large enough to hold 2*RxLen bytes
[in]RxLen- Number of data bytes to read if 0, operation effectively sets the current address
Returns
CRDSYNC_Success - Operation successful
CRDSYNC_Failure - Error - driver not opened, card not inserted or powered up
CRDSYNC_Error_BadParam - Invalid parameter, RxLen is zero, or RxBuf is NULL

)

SLE4442: Write Protection Memory
.

Note
The data must have been already stored on the card by means of crdSync_WriteData().
The contents of TxBuf is compared with the data on the card.
Only in case of match the protection bits are set.

[in]TxBuf- Pointer to buffer where the source data is stored
[in]TxLen- Number of bytes of source data
Returns

CRDSYNC_Failure - Error - driver not opened, card not inserted or powered up
CRDSYNC_Error_BadParam - Invalid parameter, TxLen is zero, or TxBuf is NULL
