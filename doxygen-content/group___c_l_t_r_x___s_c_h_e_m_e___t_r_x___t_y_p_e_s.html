#define EMV_CTLS_TRAN_TYPE_PURE_AUTHENTICATE   0x90

Gemalto/Pure: Default value for transaction type of "Application Authentication Transaction"
Special type of transaction allowing the terminal to only authenticate the contactless device, needs to be activated by Gemalto/Pure: Application flow capabilities
Value can be changed by means of EMV_CTLS_APPLIDATA_GK_TYPE::AppliAuthTransType_DFAB5A
