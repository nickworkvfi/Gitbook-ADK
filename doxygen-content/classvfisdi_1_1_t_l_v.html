Detailed Description

class for handling X690 BER TLV encoding

A TLV object consists of a tag ID and value. The value can be either a primitive value or constructed from a list of TLV objects. Whether the object contains a primitive value or other objects is indicated by a flag in the tag ID and can be queried using the TLV::constructed() method. This flag is updated according to the acual use to prevent an inconsistent setting.

Contained TLV objects can be accessed by their index or by their tag ID (plus index). If the tag ID does not exist, it is created. If the index does not exist, a new index is added.
