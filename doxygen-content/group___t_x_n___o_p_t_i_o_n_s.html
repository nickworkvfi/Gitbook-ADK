#define EMV_CT_CUST_APPLI_SELECTION_PERFORMED   0x40

B5b7 : This is a possibility that allows the app to indicate that a customer application selection has been performed. If this is set, the callback / Reentrance for candidate selection will be even called if only one candidate is left. In this special case a single candidate must not be selected automatically.
