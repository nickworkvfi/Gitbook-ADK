Initialize EMV ADK.

Explanation in programmers guide: Initialize EMV Framework

Transport tags used for serialization: CLA_EMV INS_INIT P2_SET

Author
GSS R&D Germany
Parameters
[in]numberOfAIDsNumber of AIDs
Default: EMV_ADK_DEFAULT_AIDSUPP, max.: EMV_ADK_LIMIT_AIDSUPP
TLV tag: TAG_DF3B_PARAMETER_1
[in]EMV_CallbackFunction pointer for callback (see EMV_CT_CALLBACK_FnT)
Not serialized to TLV interface.
[in]externalDataApplication data pointer, is transmitted unchanged to callback function EMV_CT_CALLBACK_FnT
Not serialized to TLV interface.
[in]optionsFlags for framework initialisation, allowed values see Options at framework initialisation
TLV tag: TAG_DF3C_PARAMETER_2
Returns
EMV_ADK_OK: success
EMV_ADK_INTERNAL: internal communication problem, illegal file content, memory allocation
EMV_ADK_PARAM: interface version mismatch or buffer overflow
EMV_ADK_TLV_BUILD_ERR: TLV processing problem
EMV_ADK_NOT_ALLOWED: concurrent call or not allowed within callback

Initialize EMV ADK.

Transport tags used in serialization: CLA_EMV INS_INIT P2_SET

Explanation in programmers guide

Author
GSS R&D Germany
Parameters
[in]numberOfAIDs... number of AIDs
TLV tag TAG_DF3B_PARAMETER_1
[in]EMV_Callback... function pointer for callback (see EMV_CTLS_CALLBACK_FnT)
no TLV tag, because it's not serialized
[in]externalData... Application data pointer, is transmitted unchanged to callback function EMV_CTLS_CALLBACK_FnT
no TLV tag, because it's not serialized
[in]options... flags for framework initialisation, allowed values see Options at framework initialisation
TLV tag TAG_DF3C_PARAMETER_2
[out]ulResult... bitstring with information flags about the outcome, see Result flags for outcome of framework initialization
TLV tag TAG_DF8F0C_INIT_RESULT_FLAGS. Can be NULL.
Returns
EMV_ADK_OK: success
EMV_ADK_PARAM: interface version mismatch
EMV_ADK_INTERNAL: internal communication problem, illegal file content, memory allocation
EMV_ADK_TLV_BUILD_ERR TLV: processing problem
EMV_ADK_NOT_ALLOWED: concurrent call or not allowed within callback
EMV_ADK_CTLS_NOT_AVAILABLE: CTLS not present on terminal
EMV_ADK_ABORT: insufficient memory to allocate main ADK context
EMV_ADK_NO_EXEC: insufficient memory to allocate other internal structs or empty desired/allowed kernel list
EMV_ADK_SAVE_ERROR: TermData/AppliData/CapKeys/Hotlist/VirtTermMap file persistence problem
EMV_ADK_READ_ERROR: TermData/AppliData/CapKeys/Hotlist/VirtTermMap or emv desired/allowed file corruption
