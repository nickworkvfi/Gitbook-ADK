◆ sysUploadBegin()

int sysUploadBegin ( const std::string &  filename,
SYSUploadType  type = SYS_UPLOAD_SOFTWARE_UPDATE,
int  dest = 1 
)

begin software or configuration package upload. (20-14)

Parameters
[in]filenamefilename including extension (mandatory in case of software update)
[in]typeconfiguration type
[in]destdisplay destination: countertop=1, EPP=2 (DFA150)
Returns
SDI error code (Result)
Note
After a successfull call to sysUploadBegin(), sysUploadData() is used to transmit the data in chunks. sysUploadEnd() completes the upload. It must be invoked also if sysUploadData() reports an error to clean up resources.
