const std::string getInternalTextKey ( unsigned  EmvTextId)

Get the requested catalog text key corresponding to the EMV text ID.

Note
A text for an EMV text ID can be composed additional multiple text keys of the catalog. For more details read describtion of UICatalog::getText(), which is used to lookup texts from the catalogs.
Parameters
[in]EmvTextIdID of the EMV Text
Returns
catalog text key or empty string TXT_NO_TXT (if SDI doesn't provide one for this EMV text ID)
