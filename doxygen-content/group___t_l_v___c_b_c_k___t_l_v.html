◆ TAG_DF76_CBK_MANIPUL_TRX

#define TAG_DF76_CBK_MANIPUL_TRX   0xDF76

Return value to EMV ADK in callback EMV_CT_CALLBACK_FnT (TAG_BF06_CBK_LOCAL_CHECKS)
Possible values:

  • EMV_ADK_ARQC to let transaction continue but force to go online
  • EMV_ADK_AAC to let transaction terminate with AAC in GENAC1
  • EMV_ADK_ABORT and any other value to let transaction abort
